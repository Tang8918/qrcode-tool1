## 🚀 第一步：部署上线（选一个平台，5 分钟搞定）

### 推荐：Vercel（最快，国内访问流畅）

1. 注册 → https://vercel.com（用 GitHub 账号登录）
2. 进入 Dashboard → 点击 **"Add New Project"**
3. 选择 **"Deploy from template"** 或直接上传文件夹
   - 方式A（推荐）：先把整个 `qrcode-site` 文件夹推到 GitHub 仓库，再在 Vercel 导入
   - 方式B：点击 **"Browse"** 直接拖入 `qrcode-site` 文件夹
4. 点击 Deploy → 自动获得 `xxx.vercel.app` 域名
5. HTTPS 自动配置，全球 CDN 加速

### 备选：Cloudflare Pages（最稳定）

1. 注册 → https://pages.cloudflare.com
2. Create a project → Upload assets → 拖入整个文件夹
3. 自动部署，免费自定义域名

### 备选：GitHub Pages（最简单）

1. 在 GitHub 新建 Public 仓库
2. 上传所有文件
3. Settings → Pages → 选择 main 分支 → Save
4. 访问 `https://你的用户名.github.io/仓库名`

### 备选：国内平台（无需科学上网）

| 平台 | 地址 | 特点 |
|------|------|------|
| 腾讯云 CloudBase | https://cloud.tencent.com/product/tcb | 国内访问快 |
| 阿里云 OSS | 开启静态网站托管 | 流量便宜 |
| Gitee Pages | https://gitee.com | 免费，国内 |

## 🌐 第二步：绑定自定义域名（可选但强烈推荐）

自有域名有助于提升 AdSense 审核通过率。

**在 Vercel 绑定域名：**
1. 进入项目 → Settings → Domains → Add Domain
2. 输入你的域名（如 `qrcode.yourdomain.com`）
3. 按提示在域名商（阿里云/腾讯云）DNS 面板中添加 CNAME 记录：
   - 主机记录：`qrcode`（或 `@`）
   - 记录值：`cname.vercel-dns.com`
4. 等待 DNS 生效（通常 5~30 分钟），Vercel 自动签发 SSL 证书

## 💰 第三步：注册 Google AdSense

### 申请流程

1. 打开 → https://adsense.google.com
2. 用 Google 账号登录 → 点击 **"开始使用"**
3. 填写网站地址（填你部署后的真实 URL）
4. 选择国家/地区（建议选**中国**或**香港**，对应银行账户）
5. 同意条款 → 提交

### Google 给你的验证代码

审核期间 Google 会给你一段代码，格式如下：

```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
     crossorigin="anonymous"></script>
```

将这段代码粘贴到 `index.html` 的 `<head>` 区域（已预留注释位置），然后重新部署。

### 审核时间

- 通常 **1～14 天** 收到结果
- 审核期间网站需保持正常访问
- 如果被拒，根据拒绝原因修改后可重新申请

### 提高通过率的要点

- ✅ 网站有真实可用的内容
- ✅ 有隐私政策页面（已生成 `privacy.html`）
- ✅ 有关于我们 / 联系方式（已生成 `about.html`）
- ✅ 网站使用 HTTPS
- ✅ 页面内容对访客有价值（本工具站完全满足）

## 📢 第四步：嵌入广告代码

AdSense 审核通过后，你会收到邮件通知。

### 操作步骤

1. 登录 AdSense 后台 → **广告** → **按广告单元**
2. 点击 **"新广告单元"**，选择类型：
   - **展示广告**（适合侧边栏 250×250 位置）
   - **文章内广告**（适合中部横幅位置）
3. 复制生成的广告代码（包含 `ca-pub-XXXXXXXX` 和 `data-ad-slot`）

### 在 index.html 中替换

打开 `index.html`，按以下步骤操作：

**① 激活顶部 AdSense 脚本**（取消注释，填入你的 ID）：
```html
<!-- 在 <head> 里找到这段，取消注释并填入真实 ID -->
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-你的ID"
     crossorigin="anonymous"></script>
```

**② 激活侧边栏广告位**（搜索 `ad-sidebar`，删除占位 div，取消注释）：
```html
<ins class="adsbygoogle"
     style="display:block;width:250px;height:250px"
     data-ad-client="ca-pub-你的ID"
     data-ad-slot="你的广告位ID"
     data-ad-format="fixed"></ins>
<script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
```

**③ 激活中部横幅广告位**（搜索 `ad-mid-banner`，同上操作）：
```html
<ins class="adsbygoogle"
     style="display:block"
     data-ad-client="ca-pub-你的ID"
     data-ad-slot="你的广告位ID"
     data-ad-format="auto"
     data-full-width-responsive="true"></ins>
<script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
```

修改完成后重新部署到 Vercel，广告会在 30 分钟内开始展示。

## 🏦 第五步：提现到银行账户

### 收款门槛

- 账户余额达到 **$100**（约 720 元人民币）后，方可申请付款

### 绑定银行账户

1. AdSense 后台 → **付款** → **付款账户** → **管理付款方式**
2. 选择 **电汇（Wire Transfer）**
3. 填写信息：
   - 收款人姓名（与银行账户一致）
   - 银行名称（如中国工商银行、招商银行等）
   - 银行账号（国内卡号）
   - SWIFT 代码（每家银行不同，百度搜"[银行名] SWIFT code"）
   - 银行地址

### 付款时间

- 每月 **21 日**结算上月余额
- 电汇到账通常需要 **3～7 个工作日**
- 到账时为美元，银行自动换汇为人民币

## 📈 SEO 加速收录

### 提交 Google Search Console

1. 打开 → https://search.google.com/search-console
2. 添加网站 → 验证所有权（HTML 文件或 DNS 验证）
3. 左侧菜单 → **Sitemaps** → 输入 `sitemap.xml` → 提交
4. 等待 Google 收录（通常 1～7 天）

### 提交百度站长

1. 打开 → https://ziyuan.baidu.com
2. 用户中心 → 站点管理 → 添加网站
3. 数据引入 → **sitemap** → 提交地址
